# Cristian Casamitjana Website

This repository contains the website files for Cristian Casamitjana.

## Files Included

- `index.html` - Main homepage
- `sobre-mi.html` - About page
- `programa.html` - Program page
- `sesiones.html` - Sessions page
- `contenidos.html` - Contents page
- `contacto.html` - Contact page
- `design-system.html` - Design system page
- `styles.css` - Main stylesheet
- `partials.js` - JavaScript components
- `assets/` - Folder containing images and other assets

## Deployment

This website is deployed using GitHub Pages. Any changes pushed to the `master` branch will be automatically deployed.

## Next Steps

1. Visit https://panoramasdmkt.github.io/cristian-casamitjana-website to view your website
2. Make any necessary changes to the HTML/CSS files
3. Commit and push changes to update the live site

## GitHub Pages Configuration

To configure GitHub Pages:

1. Go to repository Settings
2. Click on "Pages" in the left sidebar
3. Select "Deploy from a branch" as source
4. Choose "master" branch and "/ (root)" folder
5. Click "Save"
